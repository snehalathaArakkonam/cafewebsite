
# ☕ Cafe Website

A modern, responsive cafe-themed website built using Vite, HTML, CSS, and JavaScript. This project showcases a visually appealing UI with smooth navigation, clean design, and optimized performance for real-world web deployment.

## 🚀 Local Development

To run this project locally:

npm install  
npm run dev  

Then open in your browser:  
http://localhost:3000/

---

## 📦 Build for Production

npm run build  

The production-ready files will be generated in the dist/ folder.

---

## 🌍 Deployment

This project is deployed using GitHub Pages.

Steps:
1. Run npm run build  
2. Upload dist folder files to repository root  
3. Enable GitHub Pages in settings
